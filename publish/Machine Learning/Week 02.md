Jawaban akhir : Confusion Matrix (Akurasi pengambilan keputusan)

	Topic : Comparison of Machine Learning / Deep Learning Algorithms to use for Cataract Imaging

## Sets of Algorithms that can be used
ChatGPT Resources :
#### A. Traditional Image Processing + ML

1. Support Vector Machine [SVM]
	Extract features (texture, intensity, contrast, entropy) → classify with SVM.
	 Features: 
	- GLCM
	- Wavelet
	- Histogram

2. K - nearest neighbor
	 Classifies cataract by similarity between feature vectors
3. Random Forest
	 Ensemble of decision trees on extracted features.

#### B. Deep Learning Approach
1. Convolutional Neural Network [CNN]
	 Learn patterns directly from image
2. Transfer Learning [ResNet, VGG, MobileNet]
	 Pretrained Networks fine-tuned on cataract images
	 - ResNet 50
	 - VGG16
	 - DenseNet
	 - MobileNet
3. Vision Transformer [ViT]
